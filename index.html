Steps to adding to Github

  1. Create a respository in Git hub
  2. Copy the link and open the terminal
  3. git init
  4. git remote add origin (paste)
  5. git add . (adds everything)
  6. git commit -m '(message)'
  7. git push origin master

Steps to making changes the project

  1. git add .
  2. git commit -m '(message)'
  3. git push origin master

Steps to publishing the project on github for others to view

  1. In the settings, find github pages
  2. switch the source from none to master branch
  3. There will be website where the project will be live after about 10 minutes
